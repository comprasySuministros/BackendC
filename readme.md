# Supply - catalogue

Project where show the catalogue of products.

### 🚀 Enlaces

 * **Production:** https://suministros.netlify.app/api/articles

#### 🛠 Installation of project
```
npm install
```

### 🖥 Execution

```
npm run dev
```

This project start in port http://localhost:3000

### 🛠️ build with
